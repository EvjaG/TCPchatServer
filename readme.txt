list of files:
readme.txt - this file
chatserver.c - the file that contains the chat server implementation

special remarks:
Do not touch any function on the source code. Simply compile using 
"gcc -g chatserver.c -o chatserver" and run from terminal using
"./chatserver <port>" at the compiled file's location, where "<port>"
is replaced with the desired port number.

Connect to chat using "telnet 127.0.0.1 <port>" where <port> is
to be replaced by the server's port number indicated before.
